# 30. Толық жүйе (Full Project)

## Қандай технологиялар қолданылды?

Бұл жобада Java тілі қолданылды. Жоба консольдік форматта жасалды. Мәліметтерді сақтау үшін ArrayList коллекциясы қолданылды. Жоба кітапхана жүйесін басқаруға арналған.

## Жобаның архитектурасы қандай?

Жоба бірнеше класстан тұрады:

- Book — кітап туралы ақпарат сақтайды
- User — пайдаланушы туралы ақпарат сақтайды
- Library — кітаптарды басқару логикасын орындайды
- Main — бағдарламаны іске қосады

Архитектурасы:

Main → Library → Book / User

Main класында мәзір жұмыс істейді. Library класында кітап қосу, іздеу, беру және қайтару функциялары орналасқан.

## Қандай қиындықтар болды?

Жобадағы негізгі қиындықтар кітаптың статусын дұрыс басқару болды. Мысалы, кітап алынған кезде оны unavailable жасау керек, ал қайтарылған кезде қайта available болу керек. Сонымен қатар, кітапты атауы бойынша іздеу логикасын дұрыс жасау қажет болды.
