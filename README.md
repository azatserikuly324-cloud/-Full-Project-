# Library Management System

Бұл жоба 30-тақырып бойынша жасалды: Full Project.

## Тақырып

Кітапхана басқару жүйесі.

## Мақсаты

Java тілінде толық жұмыс істейтін консольдік жоба жасау. Жоба ООП принциптерін және коллекцияларды қолданады.

## Функциялар

- Кітап қосу
- Барлық кітаптарды көру
- Кітап іздеу
- Кітап алу
- Кітап қайтару

## Қолданылған технологиялар

- Java
- ArrayList
- OOP
- Console application

## Класстар

- Book
- User
- Library
- Main

## Қалай іске қосылады?

1. Main.java файлын ашыңыз
2. Run батырмасын басыңыз
3. Мәзірден керек функцияны таңдаңыз
